## b2b-order-processing-system
Enterprise B2B order processing system demonstrating domain modeling, clean architecture and scalable backend design.

## Overview 
This project simulates an enterprise B2B Order processing system where companies can place and manage large purchase orders.
The goal of the project is to demonstratete how business domains can be modeled and implemented using modern backend architeture principles.

## Business Problem 
Many companies operate with manual or poorly integrated orders systems.
This project simulates a platform responsible for:

- processing large B2B orders
- validating business rules
- managing order lifecycle
- integrating inventory and invoicing

## Key Features 
- Create and manage orders
- Order approval workflow
- Inventory reservation
- Order shipment process
- Invoice generation

## Architeture

The project follows principles of:

- Domain Driven Design
- Clean Architecture
- Separation of concerns
- Modular Monolithics

## Tech Stack

- ASP.NET Core
- C#
- Entity Framework Core
- SQL Server
- Docker
